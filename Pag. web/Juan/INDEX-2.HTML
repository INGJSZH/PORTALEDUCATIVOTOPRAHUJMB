Solucion de Errores Comunes

Encuentra soluciones rapidas a los problemas mas frecuentes en TOPRA.

  1. Verifica que el usuario y contrasena esten escritos correctamente (distingue mayusculas).
  2. Asegurate de estar conectado a la red interna del hospital.
  3. Limpia el cache del navegador: Ctrl + Shift + Del.
  4. Si el problema persiste, contacta al area de Sistemas para restablecer tu contrasena.
  1. Revisa que todos los campos obligatorios esten diligenciados (marcados con *).
  2. Verifica que el codigo del activo no este duplicado en el sistema.
  3. Confirma que tu usuario tenga los permisos de escritura habilitados.
  4. Intenta desde otro navegador (se recomienda Chrome o Edge actualizados).
  1. Revisa el rango de fechas seleccionado en el filtro; amplialo si es necesario.
  2. Verifica que el estado del filtro no este limitando las ordenes (ej: "solo pendientes").
  3. Actualiza la pagina con F5 o Ctrl + R.
  4. Si las ordenes fueron asignadas a otro tecnico, solicita acceso al administrador.
  1. Confirma que el stock disponible sea mayor o igual a la cantidad a retirar.
  2. Verifica que el servicio destino este correctamente seleccionado.
  3. Asegurate de que el insumo no este marcado como "bloqueado" por vencimiento.
  4. Reporta el error con numero de lote al area de Sistemas si continua.
  1. Cierra otras pestanas y programas que no estes usando.
  2. Verifica la conexion de red o comunicate con el area de Sistemas.
  3. Limpia el cache del navegador: Ctrl + Shift + Del.
  4. Evita horas pico (7-8 AM y 12-1 PM) para tareas que consuman muchos datos.

No encontraste la solucion? Reporta tu incidencia directamente.
